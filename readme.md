# <h3 align='left'> Usage ðŸ’» </h3>
Install all the needed packages. **Npm i** if cloning this repo, otherwise install each package manually

```
npm i

OR 

npm i node
npm i mongoose
npm i express
npm i morgan

```
Seed the database
```
npm run seed
```
Start the server
```
npm start
```
Go to localhost:3002 and app is ready to be used!
